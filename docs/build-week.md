# OpenAI Build Week 안내

이 문서는 해커톤 운영 정보를 제품 명세와 분리해 보관한다. 조사 기준일은 2026-07-15이며 일정과 규칙은 공식 Devpost 페이지를 최종 기준으로 다시 확인해야 한다.

## 개요

OpenAI Build Week는 Codex와 GPT-5.6를 활용해 프로젝트를 만드는 전 세계 개발자, 창작자, 창업자와 학생 대상의 온라인 공개 해커톤이다. OpenAI가 주최하고 Devpost에서 운영하며 전 세계 오프라인 행사도 병행한다.

- 주제: Codex와 GPT-5.6를 활용한 실제 프로젝트
- 참가 자격: 거주 국가의 법적 성년 이상, 180개국 이상 거주자 참가 가능. 일부 지역 제한
- 조사 당시 등록 규모: 약 16,500명 이상
- 총 상금: 현금 100,000달러

## 트랙

### Apps for Your Life

생산성, 창의성, 홈, 가족, 여행, 건강과 개인 금융 등 일상의 필요를 해결하는 소비자 애플리케이션.

### Work & Productivity

워크플로 자동화, 고객 지원, 분석, 영업과 백오피스 등 팀의 업무를 더 빠르고 효율적으로 만드는 도구.

### Developer Tools

테스팅, DevOps, 에이전틱 워크플로와 보안 등 개발자를 위한 프로젝트.

### Education

학생, 교사와 기관을 대상으로 교육 분야의 AI를 발전시키는 프로젝트.

Dryvre는 기본적으로 `Work & Productivity`와 가장 직접적으로 맞닿아 있다.

## 심사 기준

### Technological Implementation

Codex와 대상 모델을 얼마나 깊이 있고 능숙하게 활용했는가.

### Design

기술 PoC를 넘어 완결되고 일관된 제품 경험을 제공하는가.

### Potential Impact

구체적인 사용자층의 실제 문제를 설득력 있게 해결하는가.

### Quality of the Idea

아이디어가 창의적이며 문제 도메인에 대한 이해를 보여주는가.

강한 제출물은 문제, 해결책과 접근법을 명확히 전달하고 Codex와 GPT 모델을 사려 깊게 활용한다.

## 제출 요건

- Codex와 GPT-5.6로 구축한 작동 프로젝트
- 기능과 접근법을 설명하는 프로젝트 설명
- 3분 미만의 공개 YouTube 데모 영상
- 공개 저장소와 라이선스 또는 심사위원에게 공유한 비공개 저장소
- 셋업 방법과 구현 가이드가 포함된 README
- Codex의 역할과 GPT-5.6 사용 방식을 설명하는 문서
- 핵심 기능을 개발한 Codex Session ID
- 네 개 트랙 중 하나 선택

데모는 개발 과정에서 점진적으로 녹화하고 3분 제한을 지킨다. 저장소는 심사위원이 재현할 수 있도록 정리한다.
