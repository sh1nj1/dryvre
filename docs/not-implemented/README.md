# 구현 안함 — Backlog

이 디렉터리는 설계 참고자료와 향후 후보를 보존하지만 **현재 구현 범위에는 포함하지 않는 영역**이다.

문서가 구현 스펙 수준으로 상세하더라도 현재 코드, 데이터베이스, API 또는 UI에 적용하라는 의미가 아니다. 구현하려면 사용자가 해당 항목을 명시적으로 활성 범위로 승격해야 한다. 작업 규칙은 저장소 루트의 [`AGENTS.md`](../../AGENTS.md)를 따른다.
