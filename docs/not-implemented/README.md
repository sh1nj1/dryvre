# 구현 안함 — Backlog

이 디렉터리는 설계 참고자료와 향후 후보를 보존하지만 **현재 구현 범위에는 포함하지 않는 영역**이다.

문서가 구현 스펙 수준으로 상세하더라도 현재 코드, 데이터베이스, API 또는 UI에 적용하라는 의미가 아니다. 구현하려면 사용자가 해당 항목을 명시적으로 활성 범위로 승격해야 한다. 작업 규칙은 저장소 루트의 [`AGENTS.md`](../../AGENTS.md)를 따른다.

## 항목

- [계층적 권한 관리](hierarchical-permissions.md): 블록별 권한, 트리 상속과 서버 판정
- [페이지 모델 대안 A](page-model-dynamic-subtree.md): 모든 블록을 동적 서브트리 페이지로 여는 방식
- [페이지 모델 대안 B](page-model-folder-page.md): 중첩 없는 폴더/페이지 경계를 두는 방식
- [결정론적 문서 투영](document-projection.md): 순수 블록 모델 위에서 문서 범위를 제한적으로 투영하는 후보 설계
